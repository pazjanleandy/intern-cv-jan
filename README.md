# Intern CV – Jan Leandy S. Paz

This repository contains my Curriculum Vitae built using semantic HTML and styled with CSS as part of an internship technical task.

## Project Description
- Built using HTML5 semantic elements  
- Styled using external CSS (Box Model, spacing, and layout fundamentals)  
- No CSS Flexbox or Grid used (per task requirements)  
- Focused on clean structure, readability, and basic UI polish  

## Features
- Centered layout using `max-width` and `margin: 0 auto`
- Styled sections using padding, borders, and spacing
- Horizontal navigation using `display: inline-block`
- Project demo and documentation links
- Responsive and readable layout

## Files
- `index.html` – Main CV page  
- `style.css` – Stylesheet for layout and design
